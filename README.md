# Coconala Sales Analysis Tool (ココナラ売上分析ツール)

ココナラの売上管理画面からダウンロードできるCSVファイル（Shift-JIS）を読み込み、売上データの分析・可視化を行うブラウザベースのツールです。

## 機能概要

*   **ダッシュボード**: 売上サマリー（総売上、取引件数、平均単価）を表示。
*   **グラフ分析**:
    *   **売上推移**: 月別・日別の売上推移を表示。
    *   **サービス別売上**: 売上の多いサービスをランキング表示（Top 5/10/20）。
    *   **購入者別売上**: 購入額の多い顧客をランキング表示。
    *   **売上内訳**: サービス内訳（オプション等）の割合を表示。
    *   **曜日別売上**: 曜日ごとの売上傾向を可視化。
*   **詳細フィルター**:
    *   期間指定
    *   サービス名
    *   内訳（オプション等）
*   **データ出力 (Export)**:
    *   **CSVダウンロード**: フィルター適用後のデータをCSV形式でダウンロード。
    *   **弥生インポート形式**: 弥生会計・青色申告ソフトにインポート可能な仕訳データ形式で出力（Shift-JIS）。
    *   **freeeインポート形式**: クラウド会計ソフトfreeeにインポート可能な形式で出力（Shift-JIS）。
*   **レスポンシブデザイン**: スマートフォンやタブレットでも快適に閲覧可能。
*   **ドラッグ＆ドロップ**: CSVファイルをドラッグ＆ドロップで簡単に読み込み可能。

## 使い方

1.  **データの準備**: ココナラの「売上管理・振込申請」ページから売上CSVをダウンロードします。
2.  **ファイルの読み込み**: 本ツールを開き、CSVファイルをドラッグ＆ドロップするか、「ファイルを選択」をクリックして読み込みます。
3.  **分析**: 自動的にグラフと表が生成されます。必要に応じてフィルター条件を変更したり、集計単位を切り替えて分析します。
4.  **出力**: 「データ出力」セクションから、会計ソフトに合わせた形式でデータをダウンロードできます。

## 技術スタック

*   HTML5 / CSS3 / JavaScript
*   Bootstrap 5 (UI Framework)
*   Chart.js (Graphing)
*   PapaParse (CSV Parsing)
*   encoding.js (Character Encoding Conversion)

## 注意事項

*   本ツールはブラウザ上で動作し、サーバーにデータを送信することはありません（完全ローカル処理）。
*   Shift-JIS形式のCSVファイルに対応しています。
